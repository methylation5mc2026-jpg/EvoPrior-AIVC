## One-Paragraph Explanation

EvoPrior-AIVC is a reproducible single-cell perturbation-response prediction research repository. Its current strongest evidence is a validated residual baseline on a public Norman/scPerturb H5AD using a documented internal GEARS-compatible split. The package emphasizes data provenance, checksum discipline, leakage checks, release smoke tests, and narrow scientific claims.

## Claim Boundary

This can be shared as a public-review-ready internal GEARS-compatible Norman benchmark package. It must not be described as official GEARS, leaderboard-comparable, SOTA, biological discovery, clinical evidence, or general model superiority.
