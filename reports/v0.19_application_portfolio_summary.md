# v0.19 Application Portfolio Summary

EvoPrior-AIVC is a reproducible perturbation-response prediction project centered on public single-cell perturbation benchmarks, honest claim boundaries, and reviewer-readable artifacts.

The strongest current result is a validated residual baseline on the Norman/scPerturb GEARS-compatible internal split: MAE `0.430778`, MSE `3.668870`, Pearson `0.869224`, Spearman `0.784976` across seeds `0, 1, 2, 3, 4`.

The package includes checksum-locked data access, fixed split manifests, leakage stress checks, ablations, negative controls, model/benchmark cards, release manifests, and smoke diagnostics.

Boundary: not official GEARS, not leaderboard-comparable, not SOTA, and not biological discovery.

v0.19 adds public repository polish and reproducibility checks around this result. It is packaging and unblock documentation, not a new performance milestone.
