# EvoPrior-AIVC Mentor Review Brief

EvoPrior-AIVC is a reproducible single-cell perturbation-response benchmark package. It uses the public Norman/scPerturb H5AD (`NormanWeissman2019_filtered.h5ad`, md5 `c870e6967d91c017d9da827bab183cd6`) and a fixed internal GEARS-compatible seen0/seen1/seen2/random_combo split.

The main packaged method is `weighted_pca_ridge_s075_a10`, a PCA/Ridge residual correction over a weighted combo additive baseline. Across seeds `0, 1, 2, 3, 4`, the v0.17 validated result reports MAE `0.430778`, MSE `3.668870`, Pearson delta `0.869224`, and Spearman logFC `0.784976`. Critical leakage stress checks passed.

Reference command:

```powershell
python scripts/run_norman_residual_multiseed.py --config configs/experiment/gears_norman_v017_multiseed_residual.yaml
```

Reference output:

```text
outputs/runs/v0.17-norman-validated-residual-baseline/gears_norman_scperturb_v013/20260625T100322Z/
```

Claim boundary: this is a strong internal GEARS-compatible Norman baseline package. It is not official GEARS, not leaderboard-comparable, not SOTA, and not biological discovery.
