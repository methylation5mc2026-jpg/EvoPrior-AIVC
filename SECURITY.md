# Security

## Supported Scope

EvoPrior-AIVC is a research codebase. It does not provide a network service, authentication layer, or clinical decision system.

## Reporting

If this repository becomes public, please report security concerns through the repository issue tracker or the project owner's preferred private contact.

## Data Safety

Do not commit raw biological data, generated outputs, credentials, access tokens, virtual environments, or cache directories.

## Environment Files

Use `.env.example` as a template only. Do not commit real secrets or private paths.
